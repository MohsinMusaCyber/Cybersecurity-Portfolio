# Basic Home Network Example

## Overview

A basic home network allows multiple devices to communicate with each other and access the Internet through a router.

---

## Typical Home Network

Internet

↓

ISP Modem

↓

Router / Firewall

↓

Switch or Wi-Fi

↓

Devices

- Laptop
- Desktop PC
- Smartphone
- Smart TV
- Printer

---

## Data Flow

When a device accesses a website:

1. Device sends a request.
2. Router forwards the traffic.
3. Modem sends the traffic to the ISP.
4. The ISP routes the traffic across the Internet.
5. The website responds.
6. The response returns to the device.

---

## Cybersecurity Considerations

- Enable WPA3 or WPA2 encryption
- Change default router passwords
- Keep router firmware updated
- Disable unused services
- Use strong Wi-Fi passwords
