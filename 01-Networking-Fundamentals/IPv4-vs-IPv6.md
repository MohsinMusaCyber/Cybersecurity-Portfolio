# IPv4 vs IPv6

## Overview

Internet Protocol (IP) is responsible for addressing and routing devices across networks. The two main versions used today are IPv4 and IPv6.

---

# IPv4

IPv4 is the most widely used version of the Internet Protocol.

### Features

- 32-bit address
- Around 4.3 billion addresses
- Written in decimal
- Example:
  192.168.1.10

---

# IPv6

IPv6 was introduced to solve IPv4 address exhaustion.

### Features

- 128-bit address
- Virtually unlimited addresses
- Written in hexadecimal
- Example:
  2001:db8:85a3::8a2e:370:7334

---

# Comparison

| IPv4 | IPv6 |
|------|------|
| 32-bit | 128-bit |
| Decimal | Hexadecimal |
| Limited addresses | Massive address space |
| NAT commonly used | NAT usually unnecessary |
| Older standard | Modern standard |

---

## Why This Matters

Understanding IPv4 and IPv6 is essential for networking, cloud computing and cybersecurity because both protocols are widely used in enterprise environments.
