# OSI Model

## Layer 7 - Application
Provides network services directly to end users.

Examples:
- HTTP
- HTTPS
- FTP
- DNS

---

## Layer 6 - Presentation
Responsible for encryption, compression and formatting.

Examples:
- SSL/TLS
- JPEG
- ASCII

---

## Layer 5 - Session
Creates, manages and terminates communication sessions.

Examples:
- NetBIOS
- RPC

---

## Layer 4 - Transport
Provides reliable data delivery.

Protocols:
- TCP
- UDP

---

## Layer 3 - Network
Responsible for routing packets.

Protocols:
- IP
- ICMP

Devices:
- Routers

---

## Layer 2 - Data Link
Transfers frames between devices.

Protocols:
- Ethernet
- MAC

Devices:
- Switches

---

## Layer 1 - Physical
Physical transmission of bits.

Examples:
- Fibre optic
- Ethernet cable
- Hubs
