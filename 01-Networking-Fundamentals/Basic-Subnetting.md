# Basic Subnetting

## Overview

Subnetting is the process of dividing a larger network into smaller, more manageable subnetworks (subnets). It improves network performance, organisation and security.

---

## Why Subnetting Is Used

- Reduces network congestion
- Improves performance
- Enhances security
- Makes network management easier
- Conserves IP addresses

---

## Key Terms

### IP Address

A unique address assigned to each device on a network.

Example:

192.168.1.25

---

### Subnet Mask

A subnet mask identifies which part of an IP address represents the network and which part represents the host.

Example:

255.255.255.0

---

### Default Gateway

The device (usually a router) that allows communication with other networks, including the Internet.

---

## Example Home Network

Network:

192.168.1.0/24

Devices:

- Router: 192.168.1.1
- Laptop: 192.168.1.10
- Desktop: 192.168.1.20
- Printer: 192.168.1.30

All devices can communicate because they belong to the same subnet.

---

## Cybersecurity Relevance

Understanding subnetting helps security professionals design secure networks, segment sensitive systems and troubleshoot connectivity issues.
