# Operation 002 – Networking Fundamentals

## Objective

Develop a solid understanding of computer networking, which forms the foundation of cybersecurity.

---

## Topics Covered

- What is a network?
- LAN vs WAN
- IP Addressing
- MAC Addresses
- TCP vs UDP
- Ports and Protocols
- DNS
- DHCP
- OSI Model
- TCP/IP Model

---

## Practical Skills

- Identify private and public IP addresses
- Understand how devices communicate
- Explain the difference between TCP and UDP
- Recognise common network ports
- Understand packet flow across a network
