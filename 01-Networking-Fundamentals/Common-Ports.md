# Common Network Ports

## Overview

Network ports allow computers and services to communicate over a network. Understanding common ports is essential for networking, troubleshooting and cybersecurity.

| Port | Protocol | Service |
|------|----------|---------|
| 20 | TCP | FTP Data |
| 21 | TCP | FTP Control |
| 22 | TCP | SSH |
| 23 | TCP | Telnet |
| 25 | TCP | SMTP |
| 53 | TCP/UDP | DNS |
| 67 | UDP | DHCP Server |
| 68 | UDP | DHCP Client |
| 80 | TCP | HTTP |
| 110 | TCP | POP3 |
| 123 | UDP | NTP |
| 143 | TCP | IMAP |
| 161 | UDP | SNMP |
| 389 | TCP | LDAP |
| 443 | TCP | HTTPS |
| 445 | TCP | SMB |
| 3389 | TCP | Remote Desktop (RDP) |

---

## Why This Matters

Knowing common ports helps security professionals:

- Identify network services
- Troubleshoot connectivity
- Detect suspicious activity
- Configure firewalls
- Analyse network traffic using tools such as Wireshark
